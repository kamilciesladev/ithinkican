
# On Da Radę

Strona internetowa kampanii społecznej **„On Da Radę"** — kampanii skierowanej do kobiet, promującej świadomy wybór partnera oraz budowanie zdrowych, odpowiedzialnych relacji.

## O kampanii

Kampania porusza temat **strategicznej nieporadności** — zjawiska, w którym partnerzy unikają odpowiedzialności za dom, rodzinę i dzieci, tłumacząc się brakiem umiejętności lub doświadczenia. Jednocześnie pokazuje, że mężczyźni są w pełni zdolni do troski, zaangażowania i odpowiedzialności — na przykładach ojców i partnerów z całego świata, a także z natury.

Główne przesłanie: *Skoro inni potrafią — on również da radę.*

## Sekcje strony

- **Hero** — główny baner z logo kampanii i przyciskami nawigującymi do ulotki i spotów wideo.
- **O kampanii** — szczegółowy opis celów i przekazu kampanii.
- **Na co uważać** — lista nieakceptowalnych zachowań (przemoc psychiczna i fizyczna, brak szacunku do granic, zaniedbywanie dzieci) wraz z informacją o Niebieskiej Linii (tel. 800 120 002) i numerem alarmowym 112.
- **Dobre przykłady (Ojcowie i partnerzy z całego świata)** — przewijana karuzela kart z historiami inspirujących ojców i partnerów (np. Mark Cronin, Ben Ryan).
- **Materiały** — sekcja z zasobami kampanii, podzielona na:
  - **Spot radiowy** — odtwarzacz audio ze spotem kampanii.
  - **Plakaty/Ulotki** — materiały do pobrania lub wyświetlenia.
- **Spot wideo** — osadzony film YouTube z możliwością obejrzenia spotu na kanale kampanii.
- **Źródła** — lista źródeł i materiałów źródłowych.

## Stack techniczny

Strona zbudowana w [Astro](https://astro.build/), z komponentami `.astro` i czystym CSS (bez frameworka CSS).

## 🧞 Commands

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
